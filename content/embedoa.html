EmbedOA

Easily help patrons find Open Access articles from your Interlibrary loan pages or Libguides. No coding required.

For LibGuides

Power up your Open Access Libguide with a search bar showing patrons how to find alternatives to paywalled papers.

  1. Please sign up / sign in so we can customise this link for you and give you better service.
  2. In your Libguide, create a new box if needed. Add a title to your widget (we suggest "Search for Open Access articles").
  3. Choose to add "Media / Widget".
  4. Copy and paste the below code into the "embed code" box:
    5. <script src="{{site_url}}/static/openaccessbutton_widget.js"></script><script>jQuery(document).ready(function(){openaccessbutton_widget();});</script>
<div id="openaccessbutton_widget"></div>

    It should look like this. If it does, hit save!

  5. Now try it with the examples shown! (You may need to refresh the page.)

  6. If it doesn't work, you may need to install jquery, so try pasting this above the code you already pasted in:

    <script src="{{site_url}}/static/jquery-1.10.2.min.js"></script>

    If the syling doesn't look right you can try using the css framework we use by pasting this in too:

    <link rel="stylesheet" href="{{site_url}}/static/bootstrap.min.css">

  7. You're done. Time to tell everyone on campus!

If anything is wrong, get in touch.

For Interlibary Loan

Save time and money, and advance Open Access, by helping patrons search for alternatives to paywalled papers before requesting an ILL.

  1. Please sign up / sign in first so we can customise this widget for you and give you better service.
  2. To get started, include the following in the source code (HTML) of your page: 
    3. <script src="{{site_url}}/static/openaccessbutton_widget.js"></script><script>jQuery(document).ready(function(){openaccessbutton_widget();});</script>
<div id="openaccessbutton_widget"></div>
  3. If it looks like the version on this page, you're done! If it doesn't show up, that's okay, you probably need to include the following code: 
    4. <script src="{{site_url}}/static/jquery-1.10.2.min.js"></script>
  4. Make sure you put it above the code you put in before. If you can now see a box on the page, but it doesn't look right, add this: 
    5. <link rel="stylesheet" href="{{site_url}}/static/bootstrap.min.css">
  5. You should now be good to go! You can control where the search widget gets displayed by moving: 
    6. <div id="openaccessbutton_widget"></div>

If you have trouble getting EmbedOA working, get in touch.
