Get it

<script src="https://static.cottagelabs.com/jquery-1.10.2.min.js"></script>
<link rel="stylesheet" href="https://static.cottagelabs.com/bootstrap-3.0.3/css/bootstrap.min.css">
<script src="https://openaccessbutton.org/static/oabutton_widget.js"></script>
<script>jQuery(document).ready(function() { oabutton_widget(); });</script>

If your site already includes jquery and twitter-bootstrap css, then all you need to include is:

<script src="https://openaccessbutton.org/static/oabutton_widget.js"></script><script>jQuery(document).ready(function(){oabutton_widget();});</script>

To control where the search widget gets displayed, just put a div on the page where you want it with the ID "oabutton_widget", as follows: 

<div id="oabutton_widget"></div>
