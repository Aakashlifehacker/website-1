Safari Install Instructions

  1. Make sure your bookmarks bar is showing. If not, you can click View, and select "Show Bookmarks Bar."
  2. Click "Bookmarks" and select "Add Bookmark...". Put the bookmark on your bookmarks bar.
  3. Now, right-click the Button on the Account page and select "Copy Link."
  4. Go back to the Open Access Button bookmark in your bookmark bar, right-click, and select "Edit Address."
  5. Paste in the script from the Button and click "Done." Voila!
